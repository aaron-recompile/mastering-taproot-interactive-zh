# Jupyter Notebooks for "Mastering Taproot"

## Overview

This directory contains Jupyter Notebook versions of "Mastering Taproot", integrating theory and code examples into an interactive learning environment.

## Directory Structure

```
notebooks/
├── en/                      # English notebooks (add when ready)
│   ├── Chapter_01_Private_Keys.ipynb
│   ├── Chapter_02_Bitcoin_Script.ipynb
│   └── ...
└── shared/                  # Shared resources
    └── images/              # Image assets
```

## Quick Start

### 1. Install dependencies

```bash
pip install jupyter notebook
pip install -r ../examples/requirements.txt
```

### 2. Launch Jupyter

```bash
cd notebooks/en
jupyter notebook
```
